Guía Rápida de EOxServer¶

EOxServer es un servidor para presentar grandes archivos de datos de Observación de la Tierra (EO) y archivos de metadatos y a través de Estándares Abiertos. Se construye usando un conjunto de software de Código Abierto que incluye Python, MapServer, Django/GeoDjango, GDAL, PROJ.4 y un SpatiaLite o una base de datos de PostGIS

Esta Guía de Inicio Rápido describe cómo:

  • Use the Web Client to view, filter, subset, and download EO data
  • Use the Admin Client to inspect EO data archives

Start EoxServer¶

Choose Geospatial ‣ Web Services ‣ EOxServer. This launches a browser showing the demonstration instance available at http://localhost/eoxserver/

Work with the Web Client¶

Click the Web Client link to open the integrated EOxServer client.

Within this client you can explore the contents of the EOxServer instance. The demonstration instance is filled with ENVISAT MERIS scenes.

The client provides a set of interactions. On the top you can see a menu bar to show the Layer Selection and the Tools widgets. The main area is the map widget of the client with a set of datasets already displayed. On the bottom, you’ll find the timeslider widget, that allows you to explore the contents in the temporal dimension.

El widget de Selección de Capa le permite habilitar o deshabilitar la visibilidad y configurar el renderizado de las capas. Además, puede mostrar u ocultar la superposición de calles y seleccionar la capa de fondo. Tenga en cuenta que cuando desactiva la capa «MER FRS 1P RGB reducido», el widget del deslizador de tiempo está oculto y solo reaparece una vez que se habilita una capa de conjunto de datos de capa. El widget Herramientas proporciona tres herramientas diferentes: la Herramienta de Selección de Recuadro Delimitador, la Herramienta de Gestión de Selecciones y la Herramienta de Descarga.

De manera similar a un mapa, puede hacer zoom o mover el actual tiempo de interés visible y seleccionar un tiempo de interés dibujando en el área que contiene los puntos rojos. Tambén puede mover el cursor sobre un punto para ver el conjunto de datos ID. Haciendo click en el punto, el mapa se acerca automáticamente a la extensión del conjunto de datos.

Cuando se habilita la Herramienta de Recuadro Delimitador, puede dibujar un recuadro delimitador directamente en el mapa. Ese recuadro se utiliza para realizar consultas dentro de la Herramienta de Descarga.

Cuando se habilita la Herramienta de Descarga, se envía una consulta al servidor. Esta consulta incluye el tiempo seleccionado de interés y el cuadro delimitador. Los resultados aparecen en el widget de Descarga, donde puede seleccionar el formato, proyección y cvonjuntos de datos para descargar.

Esto fue una rápida introducción al Cliente Web EOxServer. Por favor consulte la documentación en línea para obtener más información sobre este tema.

Work with the Admin Client¶

From the EOxServer main page, click the Admin Client link and login with user admin and password admin.

El Cliente de Administración es administrador estándar de Djangos y le permite configurar los datos disponibles. Por favor, siéntase libre de explorar al cliente. Puede encontrar más información en operators” guide.

For example, you can inspect the dataset series that is loaded by default by clicking the «Dataset Series» link and then the «MER_FRS_1P_RGB_reduced» link.

Here you can also see the included datasets. You can exclude single datasets using the «delete» checkbox on the right and click on «Save».

  • Lea completamente la Documentación de Usuarios de EOxServer comenzando desde EOxServer Basics.
  • Si usted ya está en el entorno de OSGeoLive, lea la copia local de la Documentación Local de los Usuarios de EOxServer
  • ¿Listo para usar EOxServer? Entonces únete a la comunidad en listas de correo para intercambiar ideas, discutir posibles mejoras de software y preguntas.
