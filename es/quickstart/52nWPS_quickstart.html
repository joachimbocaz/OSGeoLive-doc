Guía de Inicio Rápido 52 North WPS¶

El 52 ° North WPS es un marco basado en la web de procesamiento de datos (espaciales). El servicio implementa el estándar de interfaz OGC Web Servicio de Procesamiento 1.0.

This quickstart describes how to process data and store the result in another web service using WPS operations.

Start 52°North WPS¶

To run the 52°North WPS on the Live DVD, go to Geospatial ‣ Web Services ‣ 52North ‣ Start 52North WPS or use this direct link.

Aparecerá la página de bienvenida de 52n WPS.

(If the web service is not reachable, try to start the Tomcat servlet engine following the steps at the bottom of the SOS quickstart page.)

Obtener información sobre el servicio¶

Consulte las capacidades de 52 ° North WPS con esta solicitud:

http://localhost:8080/52nWPS/WebProcessingService?Request=GetCapabilities&Service=WPS

Datos de proceso¶

  1. Click en el enlace para abrir el 52nWPS-TestClient.
  1. Make sure that GeoServer is running since the demonstration requires data from the local GeoServer installation.
  2. To do this, try to open the page http://localhost:8082/geoserver/web. If the page could not be found, select Geospatial ‣ Web Services ‣ GeoServer ‣ Start GeoServer in the menu. The GeoServer should be running after a few moments.
  3. Seleccione la solicitud de ejemplo «SimpleBuffer_out_wfs.xml» de la lista desplegable:
  1. Push the Send-Button and the request will be transmitted to the 52°North WPS which will

generate a buffer around the major roads of Tasmania with a width of 0.05 degrees and store the result in GeoServer.

  1. Copy the ResourceID from the response. This is the name of the GeoServer layer. Add the ID to the

following request: http://localhost:8082/geoserver/wms?service=WMS&version=1.1.0&request=GetMap&styles=&bbox=145.14757902405984,-43.47330297262748,148.32274986232298,-40.80286290459129&width=512&height=430&srs=EPSG:4326&format=application/openlayers&layers=Add-ResourceID-here You should get the following result:

Things to try¶

  • Pruebe otros ejemplos de solicitudes de la lista desplegable del cliente de prueba WPS.
  • Usted puede también instalar el complemento WPS para QGIS y pruébalo con el WPS de 52°North.

What next?¶

  • Cree sus propios procesos y ejecútelos usando varios clientes.
  • Configurar el WPS para ejecutar los procesos de GRASS GIS 7.
  • Configurar el WPS para ejecutar los scripts de R como procesos.

Disponible en el 52°North wiki: https://wiki.52north.org/bin/view/Geoprocessing/GeoprocessingTutorials
